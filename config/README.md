# Папка конфигурации

## Необходимые файлы:

### 1. credentials.json
Файл с учетными данными Google Cloud API для доступа к Google Sheets.

**Как получить:**
1. Перейти: https://console.cloud.google.com/
2. Создать новый проект или выбрать существующий
3. Включить Google Sheets API
4. Создать OAuth 2.0 Client ID
5. Скачать JSON файл и сохранить как `credentials.json`

### 2. token.json
Файл создается автоматически после первой авторизации.
Не нужно создавать вручную.

## Текущий статус:
❌ credentials.json - НЕ НАЙДЕН (требуется настройка)
❌ token.json - будет создан автоматически после авторизации
