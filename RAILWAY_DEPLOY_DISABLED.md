# ⛔ RAILWAY DEPLOY ОТКЛЮЧЕН

Этот проект **НЕ ПРЕДНАЗНАЧЕН** для деплоя в Railway или любые другие облачные платформы.

## Почему деплой невозможен:

1. **Локальные Windows пути** - используются абсолютные пути (`C:\Users\...`)
2. **Локальная БД SQLite** - файл `data/fiksa_database.db` на локальной машине
3. **Windows Task Scheduler** - автозапуск через планировщик Windows
4. **Локальные файлы** - Google Sheets credentials, токены, конфиги
5. **Telegram бот** - работает через `pythonw.exe` в фоне на Windows

## Что создано для блокировки деплоя:

- ✅ `railway.json` - конфиг с `builder: NONE` и выходом с ошибкой
- ✅ `.railwayignore` - файл игнорирования для Railway
- ✅ `.gitignore` - добавлена папка `.railway/`

## ✅ Правильный способ запуска:

```bash
# Локальный Windows запуск
ЗАПУСК_БОТ.bat
```

**Проект работает ТОЛЬКО локально на Windows!**
