ПРИМЕР ФАЙЛА С ЗАЯВКАМИ
======================

Скопируйте ваш Excel или CSV файл с данными о заявках СЮДА.

Формат файла:
-------------
Файл должен содержать колонки (названия могут отличаться):

• Номер заявки - уникальный идентификатор заявки
• Номер карты - номер банковской карты клиента  
• ФИО - полное имя клиента
• Телефон - контактный номер
• Адрес - адрес клиента
• Статус - текущий статус заявки
• Примечания - дополнительная информация

Поддерживаемые форматы:
-----------------------
• .xlsx - Microsoft Excel 2007+
• .xls - Microsoft Excel 97-2003
• .csv - Текстовые CSV файлы (любая кодировка)

Что произойдет после загрузки:
------------------------------
1. Фоновый сервис автоматически обнаружит файл
2. Прочитает данные и импортирует в базу данных
3. Переместит обработанный файл в папку "processed"
4. Данные будут доступны для анализа

Пример структуры Excel файла:
-----------------------------
| Номер заявки | Номер карты    | ФИО                  | Телефон      | Адрес           | Статус    |
|--------------|----------------|----------------------|--------------|-----------------|-----------|
| 12345        | 8600123456789  | Иванов Иван Иванович | +998901234567| г.Ташкент       | Активна   |
| 12346        | 8600987654321  | Петров Петр Петрович | +998909876543| г.Самарканд     | В работе  |
