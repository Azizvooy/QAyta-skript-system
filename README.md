# QAyta-skript System

Система обработки и анализа данных звонков 112

## 🚀 Быстрый старт

### Универсальный генератор отчётов (НОВОЕ!)

**Самый простой способ создать отчёт:**

```bash
# В Codespaces/Linux/Mac:
./run_report.sh

# Или напрямую:
python3 process_period_data.py
```

**Что он делает:**
1. ✅ Сканирует папку `123/` с вашими файлами
2. ✅ Автоматически определяет доступные периоды  
3. ✅ Предлагает выбрать период (или обработать всё)
4. ✅ Создаёт отчёт в `reports/`

📖 **Полная инструкция:** [ГЕНЕРАТОР_ОТЧЁТОВ.md](ГЕНЕРАТОР_ОТЧЁТОВ.md)

## Настройка

1. **Поместите файлы учётных данных** в папку `config/`:
   - `credentials.json`
   - `token.json`

2. **Загрузите данные 112** в папку `123/`

3. **Запустите генератор отчётов** (см. выше)
