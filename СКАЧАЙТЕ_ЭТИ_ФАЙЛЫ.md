## 🔨 После скачивания (3 шага):

### 1️⃣ Установка зависимостей

```bash
cd C:\QAyta-Bot
pip install -r requirements.txt
```

### 2️⃣ Тестирование (опционально)

```bash
python ОБНОВЛЕНИЕ_КАЖДЫЙ_ЧАС.py test
```

### 3️⃣ Включение автозапуска

**Нажмите ПКМ на** `ВКЛЮЧИТЬ_АВТОЗАПУСК.bat` → **Запустить от администратора**

Или просто дважды щелкните и согласитесь на запрос администратора.
